# Smart Food Waste Manager

## Summary
This project is an AI-based system designed to prevent food waste. It calculates the amount of available food and its likely expiry time, providing suggestions on how to utilize the food effectively.

## Problem
A large amount of food goes to waste, which is a major environmental and economic issue. The main reason is not knowing when the food will spoil.

## Solution
An AI model monitors the expiry date of food items and sends notifications to the user when they are about to expire. It also provides recipe suggestions using those items.

## Data
* Names of various food items.
* Average shelf life of items.
* Quantity of food items.

## Challenges
It can be difficult for the AI to accurately determine the quality of food. Users need to enter data accurately for the system to work.

## What next
In the future, this can be integrated into a mobile app, and it can automatically generate shopping lists based on food usage.
